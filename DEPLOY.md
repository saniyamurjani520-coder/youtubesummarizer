# Deploy to Vercel

1. Install Vercel CLI if needed:
   ```powershell
   npm i -g vercel
   ```
2. From the project folder, run:
   ```powershell
   vercel
   ```
3. Follow the prompts to create or link a Vercel project.
4. After deployment, open the provided URL.

Note: this project is set up for Vercel with the wrapper in api/index.py.
